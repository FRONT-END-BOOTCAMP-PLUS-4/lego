### 📘 정의

BEM(Block Element Modifier)은 CSS 클래스 명명 규칙으로, 블록-요소-수정자 패턴을 통해 스타일의 일관성과 재사용성을 높입니다.

### 🎯 주요 사용 목적

- CSS 클래스 구조화 및 충돌 방지
- 유지보수성과 가독성 향상

### 💻 코드 예시

```html
<div class="button button--primary">
  <span class="button__icon"></span>
  클릭
</div>
```

### 🧩 마무리 정리

BEM은 규모가 커질수록 유용하며, 협업 및 컴포넌트 기반 개발에 적합한 명명 규칙입니다.
