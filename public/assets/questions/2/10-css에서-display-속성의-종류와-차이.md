### 📘 정의

display 속성은 HTML 요소의 기본 박스 유형을 설정하는 속성입니다. block, inline, flex, grid, none 등이 대표적입니다.

### 🎯 주요 사용 목적

- 요소 배치 방식 설정
- 보이거나 숨기기 (`display: none`)
- 레이아웃 구현 (flex, grid)

### 💻 코드 예시

```css
.container {
  display: flex;
}

.hidden {
  display: none;
}
```

### 🧩 마무리 정리

display는 CSS 레이아웃의 출발점으로, 어떤 display를 설정하느냐에 따라 요소의 성질과 레이아웃 전략이 완전히 달라집니다.
