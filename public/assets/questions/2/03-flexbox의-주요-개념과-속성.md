📘 정의  
Flexbox(Flexible Box)는 1차원(가로 또는 세로) 레이아웃을 보다 효율적으로 배치할 수 있게 도와주는 CSS 레이아웃 모델입니다.

🎯 주요 사용 목적  
- 요소의 정렬 및 간격 자동 조절  
- 반응형 레이아웃 구성  
- 중앙 정렬, 공간 분배 등을 간단히 구현

💻 코드 예시  
```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
}
```

🧩 마무리 정리  
Flexbox는 기존 float나 inline-block보다 훨씬 직관적이며, 특히 가변 크기의 요소 배치에 강력한 기능을 제공합니다.
