📘 정의  
실행 컨텍스트(Execution Context)는 **코드가 실행되는 환경을 의미하는 추상적인 개념**입니다.  
변수, 함수 선언 등이 저장되며, 실행 흐름을 관리합니다.

🎯 주요 사용 목적  
- 스코프 체인 형성  
- 변수 및 함수의 실행 순서 관리  
- this 바인딩 결정

💻 코드 예시  
```js
function foo() {
  var a = 10;
  function bar() {
    console.log(a);
  }
  bar();
}

foo(); // 10
```

🧩 마무리 정리  
실행 컨텍스트는 JS 엔진이 코드를 어떻게 처리하는지 이해하는 핵심 개념으로, 호이스팅, this, 클로저와 밀접한 관련이 있습니다.
